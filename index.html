📌 Naming & adding new compositions (IAST-safe workflow)

Best practice: Keep display text in IAST (with diacritics), but keep filenames ASCII-only (web-safe). This avoids broken links and encoding issues.

Filename rules:

  • Lowercase only
  • No spaces (use _)
  • No diacritics (ī, ā, ṇ, etc.) in filenames
  • Example good: keeravani_geete_3.html
  • Example bad: kīravāni gīte 3.html

How to add a new composition:

  1. Add a new <li class="composition"> under the right rāga group.
  2. Fill data-type, data-raga, data-title, data-tala.
  3. Use an ASCII filename in the href, but show the IAST title in the link text.
  4. If audio exists, copy the Play button + audio div pattern and set data-src to https://www.youtube.com/embed/VIDEO_ID.
